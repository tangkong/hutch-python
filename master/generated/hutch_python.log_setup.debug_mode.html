hutch_python.log_setup.debug_mode¶

hutch_python.log_setup.debug_mode(debug=None)¶

Enable, disable, or check if we’re in debug mode.

Debug mode means that the console’s logging level is logging.DEBUG or lower, which means we’ll see all of the internal log messages that usually are not sent to the screen.

Parameters

debug (bool, optional) – If provided, we’ll turn debug mode on (True) or off (False)

Returns

debug – Returned if debug_mode is called with no arguments. This is True` if we're in debug mode, and ``False otherwise.

Return type

bool or None
