hutch_python.happi.get_happi_objs¶

hutch_python.happi.get_happi_objs(db, hutch)¶

Get the relevant devices for hutch from db.

This depends on a JSON happi database stored somewhere in the file system and handles setting up the happi.Client and querying the data base for devices.

Parameters

  • db (str) – Path to database

  • hutch (str) – Name of hutch

Returns

objs – A mapping from device name to device

Return type

dict
