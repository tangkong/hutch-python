load_conf.py¶

This module is responsible for reading and interpreting the conf.yml file. The file’s specification can be found on the Yaml Files page.

load Read the config file and convert the yaml format into a dict.
load_conf Step through the object loading procedure, given a configuration.
default_class_namespace Create a class namespace and add it to the cache.
